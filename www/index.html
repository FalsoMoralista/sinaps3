Omega centauri

Omega Centauri is a distributed cache service framework.

The main goal is to develop a distributed robust cache framework solution in
a way that developers could use it not only to reduce its servers overload but
to improve the performance of their multimidia systems.

Goals:

  • Reduce servers overload
  • Reduce connection latency between client/server
  • Improve scalability (Geographically and numerically)
